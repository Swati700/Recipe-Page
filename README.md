## 🍳 Recipe Page

A clean and responsive **Recipe Page** built using **HTML & CSS**, showcasing a simple omelette recipe with structured layout and modern design.

## 📌 Features

* 📱 Fully responsive design (Mobile + Desktop)
* 🎨 Clean and minimal UI
* 🧾 Structured recipe sections:
  * Preparation Time
  * Ingredients
  * Instructions
  * Nutrition
* 🖼️ Responsive image handling
